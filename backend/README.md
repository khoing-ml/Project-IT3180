# BlueMoon Backend

Backend API for the BlueMoon apartment management system.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm start
```

## Structure

```
backend/
├── src/
│   ├── index.js         # Entry point
│   ├── routes/          # API routes
│   ├── controllers/     # Request handlers
│   ├── models/          # Database models
│   ├── middleware/      # Custom middleware
│   └── config/          # Configuration files
├── package.json
└── README.md
```
